# **AMAZON WEB SERVICES**
AWS is a cloud computing platform which is used in the modern technology and it provides the wide range of services to individual,Enterprise and startups to build and deploy the application without the reqiurment of physical hardware setup
### it provides the wide range of services like 
- storage
- virtual servers
- databases
- machine learnig
- networking and others

Instead of purchasing the hardware setup companies rent the resources and pay only what they use

### **Traditional Method vs AWS**
| Traditional Data Center |	AWS Cloud |
|--------|---------|
| Buy physical servers | Rent virtual servers (EC2)
|Buy storage devices	| Use S3 buckets
| Buy database servers	| Use RDS
| Buy networking equipment	| Use VPC
| High upfront cost	| Pay-as-you-go

### Real-World Example

Imagine you build an e-commerce website like Amazon or Flipkart.

When a customer visits:

1. The website is hosted on EC2.
2. Product images are stored in S3 Bucket.
3. Customer details are stored in RDS (MySQL/PostgreSQL).
4. Login authentication uses IAM or Cognito.
5. Global users receive images through CloudFront CDN.
6. Application logs are stored in CloudWatch.

Everything works together.
